# Release Evidence API

A small backend-only demo app used by Cayzen sandbox runner tests.

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm test`
- `pnpm build`

## Endpoints

- `GET /health`
- `GET /releases`
- `GET /releases/:id`

The repeatable Cayzen work package for this repo is `WP-DEMO-001`: add a CSV export endpoint for release evidence without opening a pull request.
